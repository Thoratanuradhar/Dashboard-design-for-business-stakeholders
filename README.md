# 📊 Sales & Profit Analysis Dashboard

##  Project Overview
This repository contains the deliverables for **Task 4: Dashboard Design** as part of the Data Analyst Internship assignment.  
The objective was to design an **interactive dashboard** that helps business stakeholders analyze sales and profit performance.

##  Tools Used
- Power BI (for dashboard creation and interactivity)
- CSV dataset (Superstore Sales dataset from Kaggle)
- PowerPoint (for summary presentation)

##  Deliverables
1. **Interactive Dashboard**  
   - KPIs: Total Sales, Total Profit, Profit Margin, Orders  
   - Trend Analysis: Sales by Order Date  
   - Category Analysis: Sales & Profit by Category  
   - Regional Analysis: Sales & Profit by Region  
   - Top Products by Sales  
   - Slicers: Region, Category, Date for interactivity  

2. **Presentation (PPT)**  
   - Dashboard overview  
   - KPI highlights  
   - Visual analysis (trends, categories, regions)  
   - Key insights and conclusions  

##  Key Insights
- West region generates the highest profit  
- Technology category drives maximum sales and profit  
- Furniture has low profitability  
- Sales show a declining trend over time  
- Top products contribute major revenue  

##  Outcome
This project demonstrates how interactive dashboards can:
- Provide quick insights for decision-making  
- Identify high-performing regions and categories  
- Support data-driven business strategies
